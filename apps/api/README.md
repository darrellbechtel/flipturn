# @flipturn/api

Hono HTTP server for the Flip Turn MVP. Authenticates parents via magic-link
email and exposes athlete + swim + PB endpoints.

See [`docs/superpowers/specs/2026-05-04-flipturn-mvp-design.md`](../../docs/superpowers/specs/2026-05-04-flipturn-mvp-design.md) §7 for the API surface.

## Local development

```bash
pnpm dev:up
pnpm api:dev
```
